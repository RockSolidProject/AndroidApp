# Andoid climbing app
- We can track users in a climbing centers with this app installed
- Users can send messages about climbing spots
- We can detect user falls and where they were when they fell.
- Users can also detect locations of climbing holds in an image by sending it to our detection API through this app.

# Implementation
- Hold detections work through calling our detection api through HTTP
- All other comunication uses Mqtt
